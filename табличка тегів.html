Мій електронний довідник HTML
Призначення назва тегу короткий опис Стандарт
Для каркасу документа <html> Початок і закінчення HTML-документа HTML5
<head> предназначен для хранения других элементов, цель которых HTML5
<body> Определяет видимую часть документа HTML5
Для работы с текстом <p> Определяет текстовый абзац HTML5
<b> Устанавливает жирное начертание шрифта HTML5
<em> Используется для выделения из текста слова (наклонный или жирный текст) HTML5
<font color=?> Устанавливает цвет текста, используя значение цвета в виде RRGGBB. HTML5
<p align=?> Выравнивает параграф относительно одной из сторон документа, значения: left, right, justify или center HTML5
<br> Вставляет перевод строки HTML5
<i> Устанавливает курсивное начертание шрифта HTML5
<u> Добавляет подчеркивание к тексту HTML5
<strong> Выделяет текст жирным шрифтом HTML5
<br /> Переводит текст на новую строку HTML5
<hr> Добавляет в документ горизонтальную линию HTML5
Для гіперпосилань <a> предназначен для создания ссылок HTML5
<a href="#NAME"> Создает гиперссылку на помечанную часть текущего документа. HTML5
<a name="NAME"> Отмечает часть текста как место перехода по гипперссылке в документе. HTML5
<img src="imgURL"> Создает гиперссылку на рисунок, находящийся по адресу imgURL. HTML5
<a href="URL"> Создает гиперссылку на другие документы или часть текущего документа HTML5
